## Feature Highlights

### Smart Recommendations

Audiio learns your taste and suggests music you'll love:

- **Quick Picks** - Instantly generated based on recent listening
- **Weekly Rotation** - Refreshed every week
- **Because You Like...** - Similar artists and tracks
- **Mood-based Mixes** - Music for any moment

### Cross-Device Control

Control Audiio from anywhere:

- **Mobile Remote** - Use your phone as a remote
- **P2P Connection** - Works on any network
- **Real-time Sync** - Instant updates on all devices

### Customization

Make Audiio your own:

- **Theme Editor** - Full color and font control
- **Layout Options** - Compact mode, sidebar width
- **Keyboard Shortcuts** - Customizable shortcuts

### Offline Mode

Never be without music:

- **Download tracks** - Save for offline listening
- **Smart downloads** - Auto-download favorites
- **Local files** - Play your own music collection

## Getting the Most Out of Audiio

### Improve Recommendations

1. **Like tracks** you enjoy
2. **Dislike tracks** you don't want to hear again
3. **Create playlists** to train your taste profile
4. **Listen regularly** - more data = better suggestions

### Keyboard Power User

Learn these essential shortcuts:

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Pause |
| `L` | Like current track |
| `D` | Dislike current track |
| `Cmd/Ctrl + F` | Open search |
| `S` | Toggle shuffle |
| `R` | Cycle repeat modes |

See [Keyboard Shortcuts](../keyboard-shortcuts.md) for the complete list.

### Extend with Addons

Addons add powerful features:

- **More music sources** - YouTube Music, SoundCloud
- **Enhanced lyrics** - Multiple providers, translations
- **Karaoke mode** - Vocal removal
- **Scrobbling** - Last.fm, ListenBrainz

See [Addons Guide](../addons/README.md) to get started.
