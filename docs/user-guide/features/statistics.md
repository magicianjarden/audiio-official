# Statistics

Track your listening history, discover patterns, and see detailed analytics.

## Accessing Statistics

Go to **Library** > **Statistics** or click the stats icon in the sidebar.

## Export and Sharing

### Export Data

Export your stats:

1. Click **Export** in stats view
2. Choose format (JSON, CSV)
3. Select date range
4. Download file

### Shareable Cards

Create shareable images:

1. Click **Share** on any stat
2. Choose design
3. Download image
4. Share on social media

### Privacy

Stats are stored locally. To clear:

1. **Settings** > **Privacy**
2. Click **Clear Listening History**
3. Confirm deletion

## Settings

Configure in **Settings** > **Statistics**:

| Setting | Description |
|---------|-------------|
| Track History | Enable/disable tracking |
| Minimum Listen | Count plays only after X% |
| Private Mode | Pause tracking temporarily |
| Data Retention | How long to keep data |
| Scrobbling | External service sync |

## Troubleshooting

### Stats Not Updating

1. Play must exceed minimum threshold
2. Private mode may be enabled
3. Check tracking is enabled

### Missing History

1. Check date range selected
2. History may have been cleared
3. Import from backup if available

### Scrobbling Issues

1. Check addon is enabled
2. Verify service credentials
3. Check internet connection
