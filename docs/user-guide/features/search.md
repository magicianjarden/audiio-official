- Click the search bar in the header
- Press `Cmd/Ctrl + F` from anywhere
- Press `/` (slash) as a quick shortcut

## Search Types

### Tracks

Search by:
- Song title
- Artist name
- Album name
- Lyrics (if enabled)

Results show:
- Track title and artist
- Album name
- Duration
- Quick play button

### Artists

Search for artists by name.

Results show:
- Artist name
- Profile image
- Genre/style
- Follower count (if available)

Click to view artist page with:
- Top tracks
- Albums
- Similar artists

### Albums

Search by:
- Album title
- Artist name
- Year

Results show:
- Album artwork
- Artist name
- Release year
- Track count

### Playlists

Search your library playlists:

- Playlist name
- Description
- Track count

## Search Operators

### Basic Search

Just type what you're looking for:

```
daft punk
```

### Exact Phrase

Use quotes for exact matches:

```
"around the world"
```

### Filter by Type

Add prefixes to filter:

```
artist:daft punk
album:discovery
track:one more time
```

### Combine Filters

Use multiple filters:

```
artist:daft punk album:discovery
```

### Year Filter

Search by release year:

```
year:2001
year:2020-2024
```

### Exclude Terms

Use minus to exclude:

```
daft punk -remix
```

## Search Results

### Result Categories

Results are organized by type:

| Tab | Shows |
|-----|-------|
| All | Mixed results |
| Tracks | Songs only |
| Artists | Artists only |
| Albums | Albums only |
| Playlists | Your playlists |

### Sorting Results

Sort by:
- **Relevance** (default)
- **Popularity**
- **Date** (newest/oldest)
- **Name** (A-Z)

### Result Actions

Click on a result to:
- **Track**: Start playing
- **Artist**: View artist page
- **Album**: View album page
- **Playlist**: View playlist

Right-click for more options:
- Play / Play Next / Add to Queue
- Add to Playlist
- Like / Dislike
- Go to Artist / Album
- Share

## Quick Actions

### Play Top Result

Press `Enter` to play the top result immediately.

### Navigate Results

| Key | Action |
|-----|--------|
| `â†‘` / `â†“` | Move selection |
| `Enter` | Play/Open selected |
| `Tab` | Switch category |

### Inline Play

Hover over a track result and click the play button to:
- Play immediately
- Add to queue (Shift + click)

## Search History

### Recent Searches

- Shows last 10 searches
- Click to re-run search
- Clear individual or all

### Clearing History

1. Click the X next to a search, or
2. Go to **Settings** > **Privacy** > **Clear Search History**

## Search Settings

Configure in **Settings** > **Search**:

| Setting | Description |
|---------|-------------|
| Search in Lyrics | Include lyrics in search |
| Auto-complete | Show suggestions while typing |
| Safe Search | Filter explicit content |
| Search History | Enable/disable history |
| Result Limit | Max results per category |

## Search Sources

Search queries multiple sources via addons:

| Source | Content |
|--------|---------|
| Library | Your local library |
| Metadata Providers | Track/artist info |
| Local Files | Your music files |

### Provider Priority

Results are merged and ranked by:
1. Exact matches
2. Metadata quality
3. Popularity
4. Provider priority

## Tips

### Be Specific

Add more terms for better results:

```
# Less specific
love song

# More specific
love song coldplay
```

### Use Artist + Track

For common track names:

```
yesterday beatles
```

### Search Snippets

Search by memorable lyrics:

```
"we're up all night to get lucky"
```

### Keyboard Navigation

Use keyboard for faster searching:
1. `Cmd/Ctrl + F` to open
2. Type query
3. `â†“` to select
4. `Enter` to play

## Troubleshooting

### No Results Found

1. Check spelling
2. Try fewer/different terms
3. Search by artist instead of track
4. Check addon connectivity

### Wrong Results

1. Use quotes for exact phrases
2. Add artist name
3. Use type filters

### Slow Search

1. Check internet connection
2. Reduce result limit
3. Wait for addon responses
