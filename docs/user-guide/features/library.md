# Library

Manage your music collection with likes, playlists, downloads, and more.

## Accessing the Library

Click **Library** in the sidebar or press `Cmd/Ctrl + L`.

## Library Sections

### Likes

Tracks you've hearted. These influence your recommendations.

**Adding likes:**
- Click the heart icon on any track
- Press `L` while playing
- Right-click > Like

**Managing likes:**
- View all in Library > Likes
- Sort by date added, artist, or title
- Remove by clicking the heart again

### Dislikes

Tracks you never want to hear again. Disliked tracks:
- Are hidden from recommendations
- Skip automatically when encountered
- Can be reviewed and removed

**Adding dislikes:**
- Click the X icon on any track
- Press `D` while playing
- Right-click > Dislike

**Managing dislikes:**
- View all in Library > Dislikes
- Remove dislikes to restore tracks
- Optionally select reason for dislike

### Playlists

Create custom collections of tracks.

#### Creating a Playlist

1. Go to **Library** > **Playlists**
2. Click **New Playlist**
3. Enter a name
4. Optionally add a description

#### Adding Tracks to Playlists

Multiple ways to add tracks:

- **Right-click** any track > Add to Playlist
- **Drag and drop** tracks onto a playlist
- Click **+** icon in the player bar
- Use the context menu on albums or artists

#### Editing Playlists

- **Rename**: Click the playlist name
- **Reorder**: Drag tracks to new positions
- **Remove tracks**: Right-click > Remove
- **Delete playlist**: Settings > Delete Playlist

#### Playlist Options

| Option | Description |
|--------|-------------|
| Shuffle | Random playback order |
| Repeat | Loop the playlist |
| Download | Save for offline |
| Share | Generate share link |

### Downloads

Tracks saved for offline listening.

#### Downloading Tracks

- Right-click > Download
- Click download icon in player
- Download entire playlists or albums

#### Download Location

Default: App data folder

To change:
1. Go to **Settings** > **Storage**
2. Click **Change Download Location**
3. Select new folder

#### Managing Downloads

- View all in Library > Downloads
- See download size and status
- Delete individual or all downloads
- Downloads sync with library changes

### Recently Played

Tracks you've listened to recently.

- Shows last 100 tracks
- Click to replay
- Add to playlists from here

## Library Actions

### Bulk Actions

Select multiple tracks:
- `Cmd/Ctrl + Click` for individual selection
- `Shift + Click` for range selection
- `Cmd/Ctrl + A` to select all

Then:
- Add all to playlist
- Download all
- Like/dislike all
- Delete all

### Sorting and Filtering

Sort by:
- Date added (newest/oldest)
- Title (A-Z / Z-A)
- Artist
- Duration
- Play count

Filter by:
- Downloaded only
- Specific artist
- Date range

### Search in Library

Press `Cmd/Ctrl + F` while in Library to search:
- Track titles
- Artist names
- Album names
- Playlist names

## Importing Music

### From Playlists

Import playlists from other services (requires addons):
- Spotify
- Apple Music
- YouTube Music

### From Files

Add local music files:
1. Go to **Settings** > **Local Music**
2. Click **Add Folder**
3. Select your music folder
4. Files are scanned and added

See [Local Music](local-music.md) for details.

## Syncing

### Cross-Device Sync

Library syncs across devices via:
- Export/import JSON
- Manual playlist sharing

### Backup

Export your library:
1. Go to **Settings** > **Library**
2. Click **Export Library**
3. Save the JSON file

Restore:
1. Click **Import Library**
2. Select your backup file
3. Choose merge or replace

## Tips
