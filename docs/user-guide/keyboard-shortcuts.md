# Keyboard Shortcuts

Complete list of keyboard shortcuts in Audiio.

## Playback

| Shortcut | Action |
|----------|--------|
| `Space` | Play / Pause |
| `→` | Next track |
| `←` | Previous track |
| `Shift + →` | Seek forward 10 seconds |
| `Shift + ←` | Seek backward 10 seconds |
| `Ctrl/Cmd + →` | Seek forward 30 seconds |
| `Ctrl/Cmd + ←` | Seek backward 30 seconds |

## Volume

| Shortcut | Action |
|----------|--------|
| `↑` | Volume up 5% |
| `↓` | Volume down 5% |
| `M` | Mute / Unmute |
| `Shift + ↑` | Volume up 10% |
| `Shift + ↓` | Volume down 10% |

## Playback Modes

| Shortcut | Action |
|----------|--------|
| `S` | Toggle shuffle |
| `R` | Cycle repeat modes (Off → All → One) |

## Track Actions

| Shortcut | Action |
|----------|--------|
| `L` | Like current track |
| `D` | Dislike current track |
| `Ctrl/Cmd + Shift + L` | Add to playlist |
| `Ctrl/Cmd + C` | Copy track info |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Open search |
| `/` | Open search (quick) |
| `Escape` | Close search / modal |
| `Ctrl/Cmd + D` | Go to Discovery |
| `Ctrl/Cmd + L` | Go to Library |
| `Ctrl/Cmd + 1` | Go to Likes |
| `Ctrl/Cmd + 2` | Go to Playlists |
| `Ctrl/Cmd + 3` | Go to Downloads |

## Player Views

| Shortcut | Action |
|----------|--------|
| `F` | Toggle full player |
| `Q` | Toggle queue panel |
| `Ctrl/Cmd + Shift + L` | Toggle lyrics panel |
| `K` | Toggle karaoke mode |

## Window

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + M` | Minimize window |
| `Ctrl/Cmd + W` | Close window |
| `F11` | Toggle fullscreen |
| `Ctrl/Cmd + Shift + F` | Toggle fullscreen |
| `Ctrl/Cmd + R` | Refresh |

## Settings

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + ,` | Open settings |
| `Ctrl/Cmd + Shift + T` | Open theme editor |

## Queue Management

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Shift + Q` | Clear queue |
| `Delete` | Remove selected from queue |
| `Enter` | Play selected track |

## Selection

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + A` | Select all |
| `Ctrl/Cmd + Click` | Add to selection |
| `Shift + Click` | Range select |
| `Escape` | Clear selection |

## Search Results

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate results |
| `Enter` | Play selected |
| `Shift + Enter` | Add to queue |
| `Tab` | Switch result type |

## Lyrics

| Shortcut | Action |
|----------|--------|
| `T` | Toggle translation |
| `+` | Increase font size |
| `-` | Decrease font size |
| `Click line` | Seek to that point |

## Media Keys

Audiio responds to media keys on your keyboard:

| Key | Action |
|-----|--------|
| Play/Pause | Play / Pause |
| Next | Next track |
| Previous | Previous track |
| Stop | Stop playback |

Also works with:
- AirPods tap controls
- Bluetooth headset buttons
- Car audio controls

## Customizing Shortcuts

Some shortcuts can be customized:

1. Go to **Settings** > **Keyboard Shortcuts**
2. Click on the shortcut to change
3. Press new key combination
4. Click **Save**

### Default vs Custom

| Feature | Customizable |
|---------|--------------|
| Playback controls | Yes |
| Navigation | Yes |
| System shortcuts | No |
| Media keys | No |

## Tips

### Quick Search

Press `/` anywhere to instantly open search.

### Fast Volume

Hold `Shift` while pressing volume keys for larger adjustments.

### Keyboard Navigation

Use `Tab` and `Enter` to navigate without a mouse.

### Reset Shortcuts

To reset all shortcuts to defaults:
1. Go to **Settings** > **Keyboard Shortcuts**
2. Click **Reset to Defaults**

## Platform Differences

### macOS

- Use `Cmd` instead of `Ctrl`
- `Cmd + Q` quits the app
- `Cmd + H` hides the window

### Windows

- Use `Ctrl` for shortcuts
- `Alt + F4` closes the app
- `Win + ↓` minimizes

### Linux

- Uses `Ctrl` for shortcuts
- Media keys depend on desktop environment

## Accessibility

### Screen Reader Shortcuts

| Shortcut | Action |
|----------|--------|
| `Tab` | Move focus |
| `Enter` | Activate |
| `Escape` | Close/Cancel |
| Arrow keys | Navigate within components |

### Focus Mode

Press `Tab` repeatedly to cycle through:
1. Sidebar navigation
2. Main content
3. Player controls
4. Queue (if open)
