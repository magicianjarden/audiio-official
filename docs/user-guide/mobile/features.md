- Large album artwork
- Track title and artist

## Queue Management

Access the queue from Now Playing:

### View Queue

1. Swipe up on Now Playing
2. Or tap the queue icon

### Queue Actions

| Action | Gesture |
|--------|---------|
| Play track | Tap it |
| Remove | Swipe left |
| Reorder | Long-press and drag |

### Add to Queue

From search or library:
- **Tap** to play now
- **Long-press** > Add to Queue
- **Long-press** > Play Next

### Library Sections

Browse your library:

- **Likes** - Hearted tracks
- **Playlists** - Your playlists
- **Artists** - Favorited artists
- **Albums** - Saved albums

### Playlists

View and play playlists:

1. Go to Library > Playlists
2. Tap a playlist to view
3. Tap play to start
4. Use shuffle for random order

## Search

Find music from your phone:

### Quick Search

1. Tap the search tab
2. Type artist, track, or album
3. Results appear instantly

### Search Results

Grouped by type:
- Tracks
- Artists
- Albums

### Actions

Tap a result to:
- Play immediately
- View artist/album
- Add to queue

## Discovery

The home tab shows personalized content:

- Quick picks
- Recently played
- Recommended for you
- Trending

## Gestures

### Swipe Actions

| Gesture | Action |
|---------|--------|
| Swipe up on player | Open queue |
| Swipe down on player | Minimize |

| Swipe right on track | Add to queue |

### Long Press

| Element | Action |
|---------|--------|
| Track | Context menu |
| Album art | Volume control |
| Artist | Go to artist |

### Pull to Refresh

Pull down on lists to refresh content.

## Lyrics

View lyrics while playing:

1. Tap the lyrics icon on Now Playing
2. Synced lyrics scroll automatically
3. Tap a line to seek

## Performance Tips

### Battery Life

- Reduce screen brightness
- Enable battery saver
- Use shorter keep-awake time

### Data Usage

- Uses minimal data
- Album art cached
- Control messages are small (~1KB)

### Connection

- Stay on stable WiFi when possible
- P2P works on cellular
- Reconnects automatically

## Accessibility

### Screen Reader

- Full VoiceOver/TalkBack support
- All controls labeled
- Logical focus order

### Font Size

- Respects system font size
- Pinch to zoom
- Large touch targets
