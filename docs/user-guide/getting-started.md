# Getting Started

Set up Audiio and start listening to music in minutes.

## First Launch

When you open Audiio for the first time:

1. **Welcome Screen**: A brief introduction to Audiio
2. **Theme Selection**: Choose between light and dark mode
3. **Ready to Stream**: You're ready to start discovering music

## The Interface

### Main Areas

| Area | Description |
|------|-------------|
| **Sidebar** | Navigation, playlists, library access |
| **Main View** | Content area (discovery, search results, etc.) |
| **Player Bar** | Playback controls, now playing, volume |
| **Queue Panel** | View and manage upcoming tracks |

### Navigation

The sidebar provides quick access to:

- **Discover** - Personalized recommendations and trending music
- **Search** - Find tracks, artists, and albums
- **Library** - Your likes, dislikes, playlists, and downloads
- **Settings** - Customize your experience

## Finding Music

### Search

1. Click the search bar or press `Cmd/Ctrl + F`
2. Type a track, artist, or album name
3. Press Enter or click a result
4. Click play or add to queue

### Discovery Page

The Discovery page shows personalized content:

- **Quick Picks** - Tracks based on your listening history
- **New Releases** - Recently released music
- **Trending** - Popular tracks right now
- **Because You Like** - Similar to artists you enjoy
- **Weekly Rotation** - Refreshed weekly recommendations

## Building Your Library

### Liking Tracks

- Click the heart icon on any track
- Press `L` while a track is playing
- Liked tracks appear in Library > Likes

### Creating Playlists

1. Go to **Library** > **Playlists**
2. Click **New Playlist**
3. Name your playlist
4. Add tracks by:
   - Right-click > Add to Playlist
   - Drag tracks to the playlist
   - Using the context menu during playback

### Downloading for Offline

1. Right-click a track, album, or playlist
2. Select **Download**
3. Access downloads in **Library** > **Downloads**

## Playback Controls

### Basic Controls

| Action | Keyboard | Description |
|--------|----------|-------------|
| Play/Pause | `Space` | Toggle playback |
| Next | `→` | Skip to next track |
| Previous | `←` | Go to previous track |
| Volume Up | `↑` | Increase volume |
| Volume Down | `↓` | Decrease volume |
| Like | `L` | Like current track |
| Shuffle | `S` | Toggle shuffle mode |
| Repeat | `R` | Cycle repeat modes |

### Queue Management

- Click the queue icon to open the queue panel
- Drag tracks to reorder
- Right-click to remove tracks
- Clear queue with the clear button

## Mobile Remote

Control Audiio from your phone:

1. Go to **Settings** > **Mobile Access**
2. Click **Enable Mobile Access**
3. Note the connection code (e.g., "SWIFT-EAGLE-42")
4. On your phone, visit `audiio.app/remote`
5. Enter the connection code
6. Start controlling playback remotely

See [Mobile Setup](mobile/setup.md) for detailed instructions.

## Customizing Audiio

### Themes

1. Go to **Settings** > **Themes**
2. Choose a built-in theme or create custom
3. Click **Theme Editor** for full customization

### Settings to Explore

- **Audio Quality** - Set streaming quality
- **Playback** - Crossfade, gapless playback
- **Appearance** - Compact mode, sidebar width
- **Addons** - Enable additional features

## Installing Addons

Addons extend Audiio with new features:

1. Go to **Settings** > **Addons**
2. Browse available addons
3. Click **Install** on desired addons
4. Enable installed addons

Popular addons:
- **Karaoke Mode** - Vocal removal
- **Last.fm Scrobbler** - Track listening history
- **Enhanced Lyrics** - Multiple lyrics sources

## Tips for Best Experience

### Improve Recommendations

- Like tracks you enjoy
- Dislike tracks you don't want to hear
- The more you interact, the better recommendations become

### Keyboard Navigation

Learn keyboard shortcuts for faster navigation:
- `Cmd/Ctrl + F` - Open search
- `Cmd/Ctrl + ,` - Open settings
- `Cmd/Ctrl + L` - Go to library
- See [Keyboard Shortcuts](keyboard-shortcuts.md) for the full list

### Save Data

- Enable download-only mode in Settings
- Pre-download playlists for offline use
- Adjust streaming quality

## Common Questions

### Why is there no music?

Audiio streams from various sources via addons. Make sure at least one stream provider addon is enabled in Settings > Addons.

### How do I import my music library?

1. Go to **Settings** > **Local Music**
2. Click **Add Folder**
3. Select your music folder
4. Audiio will scan and import your files

### Can I transfer my playlists from Spotify/Apple Music?

Currently, playlist import is not built-in, but community addons may provide this functionality.
