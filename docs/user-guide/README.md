## Quick Tips

### Keyboard Shortcuts

- `Space` - Play/Pause
- `→` - Next track
- `←` - Previous track
- `↑/↓` - Volume up/down
- `L` - Like current track
- `Cmd/Ctrl + F` - Open search

### Getting the Best Experience

1. **Enable mobile access** to control Audiio from your phone
2. **Try the Discovery page** for personalized recommendations
3. **Customize your theme** in Settings > Themes
4. **Use keyboard shortcuts** for quick control
